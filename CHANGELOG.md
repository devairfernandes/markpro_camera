# 📋 MarkPro Camera - Changelog

Todas as mudanças notáveis para o MarkPro Camera serão registradas neste arquivo.

---

## [1.0.2] - 2026-03-09
### 🔥 Versão Oficial v1.0.2 (Lançamento Oficial)
- **🚀 Performance**: Migração de processamento de imagem para *Isolates* (o app não trava mais ao salvar a foto).
- **🎨 Branding**: Renomeado oficialmente para **MarkPro Camera**.
- **📦 Distribuição**: Pasta `/apks` integrada ao repositório para download direto.
- **🔄 OTA**: Sistema de atualização automática funcionando via GitHub.
- **📄 Documentação**: Novo README premium e manual técnico inicial.

---

## [1.0.1] - 2026-03-09
### ✨ Novas Funcionalidades
- **🖼️ Logo Customizável**: Agora o usuário pode escolher qualquer imagem da galeria como logo.
- **💾 Backup de Configurações**: Adicionado botões de Exportar e Importar via JSON (útil para transferir configurações entre celulares).
- **📍 GPS**: Melhoria na precisão das coordenadas e altura.
- **📏 Logo Size**: Aumento do tamanho da logo em 50% para melhor visibilidade.

---

## [1.0.0] - 2026-03-09
### 📦 Lançamento Beta (Versão Inicial)
- Primeiro protótipo funcional com timestamp.
- Mapa dinâmico integrado (OSM).
- Salvamento automático na galeria (Álbum MarkTime).
- Verificação de autenticidade (Original Photo Verified).
