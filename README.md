# 📸 MarkPro Camera

> **Professional Timestamp Camera for Proof of Work**

MarkPro Camera is a high-precision photographic tool designed for professionals who need undeniable proof of time, location, and authenticity. It overlays real-time GPS data, verifiable timestamps, and dynamic maps directly onto photos.

---

## ✨ Key Features

- **🛡️ Verifiable Authenticity**: Automatic "Original Photo Verified" watermark on every capture.
- **📍 Precision Geo-Tagging**: Real-time Latitude, Longitude, Altitude, and Accuracy (GPS).
- **🗺️ Integrated Smart Map**: Mini-map overlay using high-contrast vector tiles (OSM/Carto).
- **🏢 Custom Branding**: Import your own professional logo from your gallery.
- **🔄 OTA Updates**: Get the latest features automatically with our over-the-air update system.
- **☁️ Config Backup**: Export and Import your template settings via JSON codes for easy multi-device setup.

---

## 🛠️ Usage & Customization

The **"Editar Modelo"** (Edit Template) menu allows you to toggle the visibility of:
- [x] **Time & Date** (High precision)
- [x] **Full Address** (Geocoded locally)
- [x] **Mini Map** (2.5D view)
- [x] **Coordinates** (Decimal degrees)
- [x] **Altitude** (Meters above sea level)
- [x] **Your Custom Logo** (Scalable PNG/JPG support)

## 🚀 How to Install

1. Download the latest `app-release.apk` from the [Releases](https://github.com/devairfernandes/markpro_camera/releases) section.
2. Ensure "Install from Unknown Sources" is enabled on your Android device.
3. Open the APK and enjoy professional markups!

---

## 💻 Tech Stack

- **Framework**: [Flutter](https://flutter.dev/)
- **Mapping**: [Flutter Map](https://pub.dev/packages/flutter_map) (CartoDB Tiles)
- **Image Processing**: [Dart Image Library](https://pub.dev/packages/image) (Isolate-based for performance)
- **Storage**: [Shared Preferences](https://pub.dev/packages/shared_preferences)
